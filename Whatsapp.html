About Project

Project

WhatsApp Chat Analysis Project

Introduction

The WhatsApp Chat Analysis Project is a web-based application designed to analyze WhatsApp chat data and provide valuable insights in both visual and numerical formats. This project aims to help users gain a deeper understanding of their chat conversations, including communication patterns, sentiment analysis, and key statistics.


Features

1. Data Upload

Users can upload their WhatsApp chat data in text format. The application securely processes and stores the data for analysis.

2. Data Analysis

The application performs various analyses on the chat data, including: Calculation of basic statistics (e.g., total messages, average message length). Sentiment analysis to determine overall sentiment. Identification of frequently used words and phrases.

3. Visualizations

Users can visualize the chat data through interactive graphs and charts, including: Time series plots showing message frequency over time. Bar charts displaying message counts by each participant. Word clouds highlighting frequently used words. Sentiment distribution charts.

4. Insights

The application generates insightful summaries based on the analysis, helping users understand communication patterns and sentiment trends within their chat conversations.

How to Use

Upload data:

Click the "Browse File" button and select your WhatsApp chat export file.

Data Analysis:

Wait for the application to process your data.

View Insights:

Explore the visualizations and summaries generated by the application.
